JavaScript API

jQuery 在曾经一个时期内成为最流行的 JavaScript 库，它可以让开发者“write less, do more（写得更少，做得更多）”，这也是它的核心理念。通过它，用户可以更方便地处理 HTML documents、events，更轻松地实现动画效果、AJAX交互等。

尽管jQuery帮助开发者节省了大量的工作，但是并不是所有的产品都适合使用jQuery。jQuery 2.0.3版本有236KB，压缩版也有81KB，这意味着你的Web产品需要额外的加载时间和带宽，而在移动设备上则需要更多。

如果你的产品中只有少量的前端效果或DOM操作，那么完全可以抛弃臃肿的jQuery，转而使用JavaScript原生API来实现。

以下是jQuery和JavaScript实现相同操作的等价代码。

  • 选择元素

  • 创建元素

  • 添加事件监听器

  • 设置/获取属性

  • 添加/移除/切换类

  • 附加内容（Append）

  • 克隆元素

  • 移除元素

  • 获取父元素

  • 上一个/下一个元素

  • XHR或AJAX

你也可以使用其他非常轻量级的库代替，比如 microjs 列出的一些库，根据你的需求选择最合适的库，但首先要清楚，除非是不使用库无法实现某项功能，否则还是使用原生JavaScript
