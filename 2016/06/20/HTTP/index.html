HTTP

简介

　HTTP是互联网上应用最为广泛的一种网络协议。设计HTTP最初的目的是为了提供一种发布和接HTML页面的方法。通过HTTP或者HTTPS协议请求的资源由统一资源标识符（URI）来标识。HTTP完成从客户端到服务器的一系列运作流程，Web是建立在HTTP协议上通信的。

HTTP信息

状态码

  • 2XX 响应结果表明请求被正常处理

    200 OK 表示请求被server正常处理了
    204 No Content 请求处理成功，但是没有资源可以返回
    206 Partial Content 进行范围请求，然后返回范围内的内容

  • 3XX 表明浏览器需要执行特殊的处理以正确处理请求

    301 Moved Permanently 请求的资源已经被分配了新的URL
    302 Found 临时重定向 禁止POST变GET 都不遵守
    303 See Other 表示请求的资源存在另一个URL，且需使用GET方法
    304 Not Modified 允许访问，但是未满足条件
    307 Temporary Redirect 临时重定向 遵守不会从POST 变 GET

  • 4XX 表明客户端是发生错误的原因所在

    400 Bad Request 错误的请求
    401 Unauthorized 表示发送的请求需要通过HTTP认证
    403 Forbidden 请求资源的访问被服务器拒绝
    404 Not found 没有找到相应的资源

  • 5XX的相应结果说明服务器本身就有错误

    500 Internal Server Error 说明服务器执行请求发生错误
    503 Service Unavailable 说明服务器暂时处于负载或者在停机维护

HTTP首部字段

Cache-Control 控制缓存的行为
Connection 逐跳首部、连接的管理
Date 创建报文的日期时间
Pragma 报文指令
Trailer 报文末端
