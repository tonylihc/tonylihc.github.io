DOM启蒙小结

文档对象模型 (DOM) 是 HTML 和 XML 文档的编程接口。它给文档（结构树）提供了一个结构化的表述并且定义了一种方式—程序可以对结构树进行访问，以改变文档的结构，样式和内容。 DOM 提供了一种表述形式— 将文档作为一个结构化的节点组以及包含属性和方法的对象。从本质上说，它将 web 页面和脚本或编程语言连接起来了。参考链接 DOM

1、DOM (Document Object Model)：是一个由JavaScript节点对象组成的层次结构 / 树

DOM 的目的是使用JavaScript操作（删除、添加、替换、创建事件及修改）。

节点 DOCUMENT_NODE(如 window.document)、ELEMNET_NODE (如)、ATTRIBUTE_NODE(如class=”funEdges”)、TEXT_NODE
节点属性 childNodes firstChild lastChild nextSibling nodeName nodeType nodeValue parentNode previousSibling
节点方法 appendChild() cloneNode() compareDocumentPosition() contains() hasChildNodes() insertBefore() isEqualNode() removeChild() replaceChild()
文档方法 document.createElement() document.createTextNode()
元素属性 innerHTML outerHTML textContent innerText outText firstElementChild lastElementChild nextElementChild previousElementChild children
元素方法 insertAdjacentHTML

2、document.implementation.hasFeature() 探测特性 document.activeElement / hasFocus() 激活节点
isSupported() 方法也可以确定节点的实现程度信息
获取当前激活的引用 document.activeElement 相关方法 hasFocus()
全局对象: document.defaultView

3、 获取元素标签名 docuemnt.querySelector('a').tagName、docuemnt.querySelector('a').nodeName
元素属性方法 getAttribute()、setAttribute()、hasAttribute()、removeAttribute()、 hasAttribute()
类属性值列表 classList className
类属性方法 add()、remove()、toggle()、contains()

4、选取特定元素节点 querySelector() getElementById()
　选取元素节点列表 querySelectorAll() getElementByTagName() getElementByClassName()
　选择直属子元素 children
5、 offsetTop 元素上部到祖先上部的距离
offsetLeft 元素左侧边框最外缘到祖先左侧边框最外缘的距离
offsetParent 的值判定依据为查找离该元素最近的CSS定位值不为static的祖先元素。
scrollHeight scrollWidth 获取滚动元素的尺寸
scrollTop scrollLeft 获取并设置从上、左边滚动的距离
scrollIntoView() 滚动元素到视区
6、getAttribute() 获取内联CSS属性 setAttribute() 设置内联CSS属性
classList.add() 应用样式 removeAttribute() 移除样式
getComputedStyle() 获取元素的已计算样式
7、createTextNode() 创建文本节点 nodeValue() 获取文本节点值
appendData() deleteData() insertData() replaceData() subStringData() 操作文本节点
normalize() 合并兄弟节点 splitText()分割文本节点

事件

界面事件
load *加载完成时触发 unload (与前者相对)
abort 资源加载被终止 error 资源加载失败 resize 视区大小调整 scroll 滚动文档或者元素
contextmenu 右键被点击

聚焦事件
blur 失去焦点 focus获得焦点 focusin 即将取得焦点触发 focusout 即将失去焦点

表单事件
change 修改值时 reset 重置 submit 提交 select 选取文本输入框的内容时触发

鼠标事件
click 点击 dbclick 点击两次 mousedown 鼠标按下 mouseenter 鼠标移入
mouseleave 鼠标移出(不冒泡)
mousemove 鼠标移动 mouseout 鼠标移出(冒泡) mouseup 鼠标松开
mouseover 鼠标移动到元素上面

滚轮事件
`wheel(mousewheel)``

键盘事件
keydown 按键按下(不产生字符码) keypress 按键按下(能产生字符码) keyup 按键松开

触控事件
touchstart touchend touchmove touchenter touchleave touchcancel

文档相关事件
readystatechange readyState改变时触发

拖拽事件
drag dragstart dragend dragenter dragleave dragover drop

一些 method

addEventListener('事件类型',callback 回调函数,false 捕捉阶段还是冒泡阶段)
removeEventListener() 移除事件委托 preventDefault() 阻止元素的默认行为
stopPropagation() 终止事件冒泡
